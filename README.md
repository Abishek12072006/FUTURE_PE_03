# AI SEO Blog & Content Cluster Generator for Business Websites

**Internship:** Future Interns  
**Task:** Prompt Engineering – Task 3  

## Project Overview

This project demonstrates an **AI-powered SEO Blog & Content Cluster Generator** designed to help business websites create structured, search-engine-optimized content using prompt engineering.

The system generates:
- SEO-friendly blog outlines (H1–H3 structure)
- Long-form blog content optimized for search intent
- Content clusters with internal linking strategies
- Local SEO–adapted blogs (city + service targeting)

## Sample Business Used

**Matrix Salon** (Fictional Sample Client)  
A premium unisex salon based in Chennai, used to showcase how AI can support local businesses with scalable content marketing.
